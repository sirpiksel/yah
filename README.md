yah - yet another home
======================
yah is a collection of utilities

Dependencies
------------
```
dmenu
xclip
xrandr
```

Options
-------
```
h, help         print help
d, daemon       start Daemon
a, appLaunch    launch Application-Launcher
c, char         launch Charpicker
g, grab         launch Screenshot-Tool
s, scrLaunch    launch Script-Launcher
```
